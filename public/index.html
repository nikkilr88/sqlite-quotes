The Quote API

A simple random quote API perfect for FCC projects.

How does it work?

It's pretty simple. All you need to do is make a GET request to /api, and you'll get back a response with some awesome quotes.

Want to get all of the quotes without any filters?

https://quoteapi.glitch.me/api

Just want one random quote? Query params to the rescue!
default limit is 10

/api?limit=1

Looking to filter by a specific category?
motivational, funny, geek

/api?category=funny

You can even combine multiple query parameters...

/api?category=funny&limit=1

Don't See A Quote You Like?

Submit it to be added to the database! All requests will be reviewed before being either accepted or denied.
